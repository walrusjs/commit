## commit message 格式如下

```
<type>[optional scope]: <description>

[optional body]

[optional footer(s)]
```

完整示例

```
feat(login): 完成登录功能

添加登录表单、联调登录接口...

Reviewed-by: Tom
Refs #133
Close #100
```
